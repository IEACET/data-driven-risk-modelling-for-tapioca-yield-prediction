# Data-Driven Risk Modelling for Tapioca Yield Prediction

## 📝 Description
To develop a data-driven yield risk modelling framework that analyses the relationship between climatic stress factors and tapioca yield, enabling the identification and quantification of conditions associated with yield reductions at the block level.
